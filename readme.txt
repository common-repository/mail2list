Mail2List let you send html mails to the users of your blog or to a mailing list.

== Description ==

With Mail2List you can manage your websites mailing lists. Main functions are:

* Set up different mailing lists
* Use default generated mailing lists (administrators, subscribers...)
* Add custom filtered lists
* Manage sent mails
* Edit your mail body html as you want
* Check existence of mail addresses
* Ajax email send for big mailing lists
* Edit your mail template

**New in this version: 1.2**

* Solved some bugs
* Added compatibility with WP 2.9.x


**New in this version: 1.3**

* Solved broken 1.2 version

== Installation ==

1. Upload `Mail2List` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
